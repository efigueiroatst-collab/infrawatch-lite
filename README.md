# InfraWatch Lite

Ferramenta de monitoramento em tempo real desenvolvida em Python.

## 📊 Funcionalidades

- Monitoramento de CPU
- Monitoramento de memória RAM
- Monitoramento de disco
- Atualização contínua dos dados
- Interface web simples para visualização

## 🛠️ Tecnologias

- Python
- psutil
- HTML

## ▶️ Como executar

1. Instale a biblioteca necessária:
pip install psutil

2. Execute o script:
python monitor.py

3. Abra o arquivo index.html no navegador

## 📌 Objetivo

Projeto desenvolvido com foco em aprendizado prático em monitoramento de sistemas e infraestrutura de TI.
